## Swordfish
A demo project for Next 13 `/app` directory

> It's still a work in progress. Some things may not work.

### Using
- i18next
- Tailwind
- Prisma
- Next-Auth
- nodemailer
- mjml

### How to run
1. `pnpm i`;
2. `cp .env.example .env.local` and fill it;
3. `pnpm dev`
4. open `http://localhost:3000`
